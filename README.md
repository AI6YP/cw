# CW Key

## Features

  * Autimatic
  * USB connected
  * [Makey Makey](https://github.com/sparkfun/MaKeyMaKey)
  * Browser JS
  * PTT key
  * Capacitive touch paddle ENIG (Electroless Nickel Immersion Gold) finish
  * https://hackaday.io/project/19129-tiny-cw-capacitive-touch-paddle
  * Rotary encoder
    - speed
    - volume
    - VFO
